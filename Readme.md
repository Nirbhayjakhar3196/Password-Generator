# 🔐 Password Generator App

A modern and responsive Password Generator built using **HTML, CSS, and JavaScript**.

This project generates secure random passwords with customizable options like uppercase letters, lowercase letters, numbers, and symbols.

---

## 🚀 Features

✅ Generate random secure passwords  
✅ Copy password to clipboard  
✅ Password strength indicator  
✅ Increase / decrease password length  
✅ Range slider for password length  
✅ Auto-generate password on refresh  
✅ Reload button to generate new password  
✅ Animated password change effect  
✅ Responsive and modern UI  

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## ⚡ How It Works

1. Select password options:
   - Uppercase
   - Lowercase
   - Numbers
   - Symbols

2. Choose password length.

3. Click reload button 🔄 to generate a new password.

4. Copy password using the Copy button.
